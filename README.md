# WEBCAM
"Build a model that detects hand gestures via webcam and converts them to text or speech in real time. Uses MediaPipe to extract 21 hand landmarks and a lightweight MLP classifier to recognize ASL/ISL alphabets. Output letters accumulate into words, which can be spoken using text-to-speech.
